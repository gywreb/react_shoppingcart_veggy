# VEGGY - SHOPPING CART

A Shopping Cart Web build with React, Redux, Material-UI

Reference from proj : [sivadass/react-shopping-cart](https://github.com/sivadass/react-shopping-cart)

Demo : [gywreb-veggy](http://gywreb-veggy.surge.sh/)

- Install:

```
  yarn install
```

- Start

```
  yarn start
```

- Build

```
  yarn build
```
